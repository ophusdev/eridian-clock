### Eridian Clock - Project Hail Mary

![Project Screenshot](https://raw.githubusercontent.com/ophusdev/eridian-clock/refs/heads/main/public/eridian-clock.png)

A real-time, aesthetically minimalist application built with React and TypeScript that displays and synchronizes time between **Earth Standard Time (UTC)** and a custom **Eridian Standard Time**.

---

### Prerequisites

* Node.js (LTS recommended)
* npm or yarn

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/ophusdev/eridian-clock.git
    
    cd eridian-clock
    ```

2.  **Ensure dependencies are installed:**
    ```bash
    npm i
    ```

3.  **Run:**
    ```bash
    npm run start
    ```
